== Description ==

Add new order status to WooCommerce and set default status for new order.<br >
Call restfull API's after change order status (Optional).<br >
Send email to customer after change order status (Optional).<br >
Check restfull apis response as in post section (Optional).


== Installation ==

1. Log into WordPress
1. Select Plugins, then "Add New"
1. Select Upload Plugin
1. Choose the zip file, then Install and activate
